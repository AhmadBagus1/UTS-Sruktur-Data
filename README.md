Sistem Manajemen Data dan Penggajian Karyawan (C++)
Deskripsi Proyek

Program ini merupakan aplikasi manajemen data karyawan berbasis C++ yang dibuat menggunakan konsep Object-Oriented Programming (OOP). Aplikasi mampu mengelola data karyawan mulai dari menambahkan data, mencari data berdasarkan NIP, mengubah data, hingga menghitung dan menampilkan total gaji setiap karyawan.
